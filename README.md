# Sales-Data-Analysis-Dashboard-using-Excel
This project analyzes retail sales data to identify key revenue trends and product performance.  An interactive dashboard was built using Excel Pivot Tables, slicers, and charts to visualize sales metrics such as revenue by region, product category, payment methods, and monthly sales trends.
